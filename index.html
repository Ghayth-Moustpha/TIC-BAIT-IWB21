مقدمة عن HTML

HTML (HyperText Markup Language) هي لغة توصيف النصوص الأساسية لبناء صفحات الويب. بدأت HTML في التسعينيات وساهمت في تطوير الإنترنت كما نعرفه اليوم.

تاريخ HTML

  • 1991: ظهور HTML 1.0
  • 1995: HTML 2.0 تم توسيعها لتشمل الجداول والنماذج
  • 1997: HTML 3.2 أضافت دعم القوائم والبرمجيات
  • 1999: HTML 4.01 — دعم CSS وDOM
  • 2000: ظهور XHTML 1.0 لتحسين التوافق مع XML
  • 2008: HTML5 تم تطويره ليصبح أكثر مرونة ودعماً للوسائط الحديثة

XHTML

XHTML (eXtensible HyperText Markup Language) هو نسخة صارمة من HTML مبنية على XML. تم تصميمه لضمان التوافق مع معايير XML وتسهيل معالجة الصفحات بواسطة الآلات.

إصدارات XHTML

  • XHTML 1.0: 3 أنواع - Strict، Transitional، Frameset
  • XHTML 1.1: نسخة أكثر صرامة وحدات Modular
  • XHTML 2.0: لم يعتمد رسمياً وتم إلغاء المشروع
  • XHTML5: موازي لـ HTML5 مع قواعد XML

مميزات XHTML

  • ✔ صرامة أكبر في كتابة الوسوم
  • ✔ كل الوسوم يجب أن تكون مغلقة
  • ✔ خصائص الوسوم يجب أن تكون بين علامات اقتباس
  • ✔ التوافق مع XML وXSLT
  • ✔ صفحات أكثر نظافة وسهلة المعالجة آلياً

مقارنة بين HTML و XHTML

المعيار HTML XHTML
المرونة مرنة وتتسامح مع الأخطاء صارمة، الأخطاء تمنع العرض
إغلاق الوسوم اختياري في بعض الوسوم مثل <br> إلزامي لجميع الوسوم، مثلاً <br />
حالة الأحرف يمكن استخدام الكبيرة أو الصغيرة يجب استخدام lowercase لجميع الوسوم
التوافق المتصفحات تتجاهل الأخطاء الأخطاء تمنع عرض الصفحة أو تتسبب في توقف التحليل

مثال كود: HTML يسمح به وXHTML لا يسمح به

في HTML يمكنك كتابة الوسوم بدون إغلاق أو بدون علامات اقتباس:

<br>
<img src=example.jpg alt=صورة>

بينما في XHTML يجب أن تكتب بالشكل الصارم:

<br />
<img src="example.jpg" alt="صورة" />
